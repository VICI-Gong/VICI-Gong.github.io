mac 终端配置

程序员需要经常与终端打交道，好的配置及视觉效果能提升工作效率，现将工作中用的终端配置总结如下。

iTerm2

去https://www.iterm2.com/下载dmg安装包安装即可。

Homebrew

官网在这里，安装命令如下：

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Sublime Text 2

官网在这里，下载安装即可。

终端配置

终端配置主要是包括bash配置，视觉配色，vim配置等等，之前都是手动一项项去配。偶然之间发现有大神已经写好自动化脚本了。git地址：

https://github.com/mathiasbynens/dotfiles

配置命令：

git clone https://github.com/mathiasbynens/dotfiles.git && cd dotfiles && source bootstrap.sh

虽然说能够达到自动化配置了，但还是建议看看bash脚本，了解配置过程中涉及到的点，方便自己添加一些定制化的功能。
